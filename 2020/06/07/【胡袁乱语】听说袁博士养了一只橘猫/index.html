【胡袁乱语】听说袁博士养了一只橘猫

GMI哨向AU系列
简介： 光鸣塔的成员都发现黑暗向导袁博士在宿舍养了一只宠物橘猫。
隐婚文学，一发完

私设：

  • 哨向有两级精神体。一级精神体是公开的，与个人的天赋和战斗能力挂钩。二级精神体是后期演化的，属于哨向隐私范畴，一般只对信任的亲人朋友公开。大部分哨向在分化时一级精神体就会出现，并且陪伴主人参与战斗。分化至少十年以上二级精神体才会觉醒，长期陪伴主人生活。大多数哨向的二级精神体是一级精神体的幼崽模式。
  • 黑暗哨兵和黑暗向导可以独立执行任务，没有结合热。

1.
最先发现袁广泉养猫的是何宜霖。

何宜霖是袁广泉的前室友。两人是光鸣塔里公认的中生代最强向导。何宜霖是几十年不遇的全频段精神操控与破坏的高手，被不堪其扰的战友们戏称光鸣一霸。而袁广泉则是更为稀少的黑暗向导，擅长中远程精神暗示与攻击的同时，物理输出又能媲美普通哨兵。他们与塔里的全能哨兵何亮辰和黑暗哨兵胡浩组成了海四团，经常承接塔里的高危任务。

在海四团的新任务下发当日，何宜霖和袁广泉约在了高阶向导练习室碰面。高个向导本来想对他的前室友来个例行物理控制，结果胳膊抬起一半就往后退了几步，接连打了几个大喷嚏。
“你离我远一点。”何宜霖抽了抽鼻子，感觉自己的过敏性鼻炎又要犯了，“你是刚撸了名宇的狗崽子吗，怎么裤腿上全是黄毛。”他想到了黄名宇那只爱围着人转的黄毛宠物柴犬。狗崽子长得就像主人的一级精神体翻版，导致这位控制达人经常因为撸错狗而狂打喷嚏。

“对不起，”袁广泉抱歉地笑了一下，“家里的橘猫太粘人了。”

“我们刚不一起住，你就养宠物凹。”何宜霖拖着懒洋洋的腔调嘟囔着，“单人宿舍多小凹，你的切不拉什卡个头那么大，不会欺负它吗。”他想到了袁广泉那只硕大的北极熊精神体，开始担心这只没见过的橘猫，生怕北极熊一个没注意把这只可怜的小动物坐成屁垫。

有别于一般向导温和无害的小型动物精神体，切不拉什卡不论从体型还是武力上都威猛了太多，跟温和秀气的留俄博士袁广泉本人反差极大。作为他的前室友，何宜霖每天早上一睁眼就能看见挂在单人床边缘摇摇欲坠的袁博士，和霸占了整个床还抢被子的北极熊。每到这个时候，他才觉得自己的一级湾鳄和二级短吻鳄都相当贴心，虽然性格都挺恶劣，但是至少冬天不抢被子，夏天还能当凉席。

“切不拉什卡还挺喜欢加菲的。”袁博士从抽屉里拿出了一个长相别致的滚毛器，仔细地滚起了裤腿，“而且我的二级精神体刚刚觉醒了，切不拉什卡最近也不怎么出现了。”说罢，一只北极熊幼崽凭空出现，挂在了何宜霖脖子上，欢快地薅着他的头发。“来，给你介绍一下喀秋莎。”袁博士一开口，小熊抱的更紧了，还开心地啃了两口。

“我发现你的精神体一级比一级恶劣。”何宜霖一边躲避着小熊的攻击，一边还试图偷摸几把熊脑袋。对于一个患有过敏性鼻炎的毛绒控向导来说，撸毛绒绒的精神体比撸真实的宠物实在是愉快太多。“熊崽子快收起来凹，今天有任务，我们要去先哨兵所跟亮辰和浩哥汇合。”

袁广泉打了个响指，小白熊乖乖地消失在他身体里。何宜霖整了整发型，放出了自己的湾鳄。一人一鳄随后迈着六亲不认地步伐走向大门。结果没走两步，就听到湾鳄凄厉的嘶吼。他回头一看，切不拉什卡正懒洋洋地趴在大鳄鱼身上翻滚。

“起来凹大屁股！说了多少次了阿密特不是凉席！”何宜霖感觉战斗还没开始他就已经暴躁了。袁博士真诚而无辜的摊了摊手，说道：“你知道的，非战斗期间，我管不了切不拉什卡。你的阿密特确实比较凉快。”

经过一番折腾，何宜霖艰难地扛着阿密特往哨兵所走去。看着前面步履轻快的袁博士和切不拉什卡，他第n次觉得，精神体代表向导阴暗的内心这个伪科学说得如此有道理。

2.
“广泉，何大suo你养猫了！”在任务顺利完成后，何亮辰成了第二个知道加菲的人。

“是的，养了一只橘猫。”袁广泉一边专心地开车，一边轻声回复着后排兴致勃勃的哨兵，“豆豆你小声一点，浩哥今天输出有点多，你让他休息一下。”他伸出了一些精神触丝，默默地给副驾上小憩的黑暗哨兵屏蔽了周围的噪音。

“它叫什么名字？我能去看看它嘛。”年轻的哨兵压低了声音，但是还是掩盖不住声音里的雀跃。“它是不是比浩哥的泰戈尔还可爱。”

何宜霖翻了个白眼。泰戈尔是在车尾的精神体车厢里正在打鼾的那只东北虎。在今天在战场上，泰戈尔不到十秒就击垮了敌人的阵型，一声咆哮就吓死了几十个低级精神体，跃入敌方没几分钟便生撕了好几个高阶精神体猛兽。这样一个顶级精神体，何宜霖实在不觉得它跟可爱能扯上半点关系。可是他的结合哨兵习惯性眼瞎能怎么办呢。

“它叫加菲。”袁广泉回答道，“它是只自由的小猫，与其说是宠物，不如说是我的新室友。如果你运气好的话，应该能在我宿舍里见到它。”他透过后视镜快速地看了一眼泰戈尔，猛虎被他的北极熊拱到了角落里，随着车辆的颠簸翻了个身，毛绒绒的虎掌搭在北极熊温暖的皮毛上，一副很舒服的样子。这样温馨的场景让他不禁想到了喀秋莎抱着加菲午睡的画面，顿时感觉心中一片柔软。

“都挺可爱的。”他笑着说。何亮辰一听，眼睛都亮了起来，心中已经开始默默计划他的撸猫之旅了。

“浩哥的小泰戈尔也挺可爱的。”他愉快地回忆着胡浩的二级精神体，“我看过他抱着小老虎的照片，小小的一只，还在他怀里打哈欠。可惜浩哥从来不肯放出来给我撸。”

何宜霖看他对别人的精神体和宠物那么上心，浑身又开始冒酸气。“怎么，这会儿嫌弃阿密特和史耐皮不好摸了？夏天还赖着蹭凉席，抱着史耐皮不撒手，一结合就觉得别人家的精神体更好摸了凹。”

“我还没说某人的两只瓜鳄鱼总是要吞我的佩奇呢！”何亮辰满脸通红地回着嘴。自从他和何宜霖结合以后，对方的两只精神体总是变着法的折腾他的二级精神体佩奇，一只可爱的小香猪。连何宜霖本人都时常抱着佩奇使劲撸，还仗着身高优势把佩奇举到何亮辰抓不到的位置，气的他跟佩奇哼哼的声音都同步了。“史耐皮原来挺乖的，现在跟你一样恶劣！”他一激动，吵醒了后排他的一级精神体小鲲。小鲲是一只大翅鲸。感受到主人心中的不爽，小鲲大嘴一张，立刻给何宜霖当头来了场海水浴。阿密特见状也一口咬上了小鲲的尾鳍。后座两个一米八几的大男人和精神体车厢两只巨型海生动物小学鸡一般地扭打起来。场面一度非常混乱。

袁广泉刚想投射几个精神暗示制止一下后座两位不靠谱的朋友，后排的东北虎突然咆哮起来。正在拉扯的一对哨向也感受到了一股强势的精神威压，慢慢停手了。副驾上的胡浩依然闭着眼睛，但是显然已经被两人吵醒。
“要打关门回被窝里打。”胡浩不耐烦地说，“我的泰戈尔是个猫科动物，不吃狗粮！”

3.
第三个知道橘猫的人是黄名宇。

年轻的向导还没法完全控制自己的一级精神体，最近总在粘着黑暗向导加强训练。袁博士脾气很好，不论是黄名宇还是同期的王嘉欣和方晓东，只要弟弟们有问题，他都认认真真地解答。没有任务时，他甚至会在训练场陪到深夜才回宿舍。

然而几个年轻向导的精神体都有些害怕北极熊，训练时很少敢跟切布拉什卡一起出现。这导致黄名宇已经练了几个月了，然而他的精神体柴犬阿黄还是一副自由散漫的样子，他不过在训练间隙跟他的半绑定哨兵王上多聊了两句，一回头，阿黄就又不见了。无奈之下，他只能跟本日的教官告假，在塔里各处伸出精神触丝召唤着阿黄。追着细微的精神波动，他一路走到了袁广泉的宿舍。

宿舍大门紧闭，阿黄的叫声却从门缝里传了出来。黄名宇敲敲门，没有人应答，本想给袁广泉发个信息求助，结果门突然自己打开了。

一只神气的橘猫从门把手上跳下来。它毛绒绒的，身子略微有些肥胖。一双金色的眼睛又圆又亮，瞳仁在阳光的照射下缩成一条窄缝，让人感觉既风情又危险。橘猫迅速地扫了他一眼，然后闲庭信步地走回了窗台下的小沙发上晒太阳。袁广泉不在房间里，而阿黄却在沙发旁边围着橘猫一边叫一边转圈圈，完全没有理会黄名宇的意思。橘猫偶尔懒洋洋地回应了柴犬两声，不时还拿爪子拍拍它的脑袋。黄名宇在门口看着，莫名有一种老师在训导学生的感觉。

他蹲在门外看一猫一狗互动了约么半小时，并没有等到下班的袁广泉。橘猫伸出前爪大大地来伸了个懒腰，然后跳下沙发，带着阿黄走到了黄名宇身边。他本想训斥阿黄几句，却听到橘猫一声不满地咕噜声。阿黄欢快地蹭了蹭他的裤腿，然后小跑进到他的精神域里。

黄名宇想拍拍橘猫毛绒绒的脑袋，但却被它躲过了。橘猫不满地拿它的肉垫打了他几下，仿佛在示意他没事快走。他对着这只骄傲的小动物说了声谢谢，然后关门离开了。

之后的几天里，黄名宇明显感觉到阿黄听话了许多。他和阿黄的配合训练也越来越顺利了。他将这个奇闻告诉了王上，却看对方皱起了眉头。

“你确定那是袁博士的宠物吗？你是不是见到了他的二级精神体。阿黄的样子明显是被高阶向导训练过了。”

“我见过Q的二级精神体，是只小号切布拉什卡。”黄名宇回复到，“这只橘猫肉垫的触觉不像精神体啊。而且宜霖哥说他会因为Q身上的猫毛打喷嚏的。精神体是不会引起这种反应的。我觉得应该就是他的宠物。”

王上还是觉得哪里怪怪的。正准备发挥学霸精神继续刨根问底，结果接到了塔里发来的任务通知。他要带着初级向导黄名宇执行一项中级任务，黑暗哨兵胡浩会在一边临场指挥。他放出了自己的鳄龟杰尼，然后带着黄名宇和阿黄去哨兵所跟胡浩会和。

胡浩正靠在塔指派的任务车副驾上假寐，泰戈尔也伏在他的脚边呼噜着。一人一虎看起来都有些疲惫的样子。黄名宇莫名觉得这场景有点熟悉。

“猫科动物睡觉的样子都挺萌的。”他跃跃欲试地搓着手，低声对王上说着。毛二的雪豹和Q的橘猫睡觉的时候也这么可爱。

“可爱也不能摸。”王上拍掉了他不安分的手，让他赶紧上车，顺便把杰尼和阿黄赶到精神体车厢去。王上自己坐上了驾驶座，还没坐稳，胡浩便睁开眼跟他打了个招呼。泰戈尔摊开前爪大大地伸了个懒腰，慢慢踱步到精神体车厢里。

去往目标地点的路上，王上一边开着车，一边听着黄名宇一路在向胡浩请教各种问题。胡浩都一一耐心的解答，有时甚至还让后车厢的泰戈尔给阿黄做起示范来。杰尼在一边趴着，偶尔也给阿黄做做导师。

“浩哥，”王上插嘴问道，“你会允许其他人的二级向导住到你的房间吗？”

“不会。”胡浩回答说，“哨兵只会允许结合向导的二级精神体长期在自己的私人领域逗留。我是黑暗哨兵，不需要结合向导。”

“那黑暗向导呢？”王上追问道，“你觉得袁博士会把别人的二级精神体养在自己房间吗？”

胡浩似乎犹豫了一下，“我想应该不会。黑暗向导也不需要结合哨兵。我与他只在最高级别的行动中才做哨向配合。日常任务一般是辰和宜霖配合，我俩单独行动。”他顿了顿，又说道“我想他应该也没法容忍其他人的二级精神体长期入侵他的私人空间。”

“上哥，我都说那是Q养的宠物猫了。你就不要再纠结了。”黄名宇把头探到前排说到。

4.
董攀第四个发现了橘猫。

作为精通中远程精神暗示领域的向导，董攀的特训课程很多都跟袁广泉重合。在他找到自己的结合哨兵郭虹旭以前，他也和作为黑暗向导的袁广泉完美的配合过几次任务。当他第三次看到袁博士手上的抓痕和脖子上的创可贴时，他觉得有必要关心一下老搭档的私人生活。

“哥，你是找了个凶残的对象吗？”下班路上，董攀毫不含糊的打了直球。

“猫抓的。”袁博士温柔地笑了笑。“家里的猫脾气比较大，我在外面摸了其他动物，他也会吃醋的。”

“猫那么凶，喀秋莎不会跟他打架吗？”年轻向导又追问到。

“不会的，加菲很可爱。喀秋莎喜欢抱着他睡觉。”他说着，拿出了手机，给董攀看了猫的照片。一只橘猫正躺在雪白色的毛垫上晒太阳。董攀仔细地端详了一下，发现那团毛垫是袁广泉的切布拉什卡。

“是挺可爱的。也难怪你和你的两个精神体都变成猫奴了。我能去你宿舍里撸猫吗？”董攀正说着，一旁突然跑出一只红色的小动物，一把抱住了袁广泉的大腿。他低头一看，只见董攀的一级精神体，小熊猫煎饼也睁着同样的大眼睛盯着他。袁广泉被这群一大一小盯的心都化了，随口应了下来。

“我建议你带上旭旭和毛二。”袁博士冷静下来后，还是积极地为自己寻找后路。“不然明天你我都会挂彩。猫科动物之间会更亲近一些。”

第二天，一只小熊猫带着一只云豹和一只雪豹敲开了袁博士的宿舍门。橘猫斜眼瞟了几个年轻人，然后又傲娇地转过头对着窗户吹风。两个大型猫科动物胆怯地蹲地离它远远的，只有董攀的小熊猫欢快的扑到了橘猫身上，迎来了一通喵喵拳。

三个年轻人有些惊奇地围在沙发旁边看着自己的精神体与橘猫之间的互动。站在猫尾巴边上的哨兵毛二突然大笑了起来。

“你们觉不觉得，加菲长得像我哥。小脑袋大屁股。趴着的样子跟泰戈尔一模傻哈哈哈。”

董攀和郭虹旭听完惊恐地对视了一眼，好像撞破了什么大秘密。郭虹旭默默地给他的云豹小桂花下了个暗示。小桂花战战兢兢地往前迈了两步，拿脑袋蹭了蹭橘猫的下巴，然后立刻躲回了郭虹旭的精神域里。两个小情侣在精神域里共享了一下小桂花的触觉，然后立刻拖着毛二告辞了。

“我还没撸猫呢！”毛二回去的路上不满地抱怨着。

“你先考虑明天的哨兵特训怎么不被你哥摁在地上摩擦比较现实。”郭虹旭默默叹了口气，小桂花恨铁不成钢地拿着大尾巴抽了毛二的雪豹一把。

“广泉哥不愧是俄罗斯回来的男人，”董攀在精神域里对郭虹旭感慨着，“果然喜欢喝最烈的酒，找最野的男人。”

“希望他以后不会在最贵的ICU抢救。”郭虹旭心有余悸地接上了下一句暗号。

“让我们祝福广泉哥。”年轻的哨向在心里默默为袁博士祈祷着。

5.
后来，越来越多的人看到了袁博士的橘猫。他时不时地会在塔的各个角落遛弯。有时候眯着眼睛被何亮辰抱在怀里顺毛。有时候趴在何宜霖的大衣上睡觉。更多时候，他似乎在带着年轻哨兵向导走失的精神体训练。毛二的雪豹每次见到它，都乖顺地翻开肚皮任它舔毛。

起初大家都觉得这些精神体是服从于袁博士的猫奴精神体切布拉什卡。直到有一天，哨兵所的所长，黑暗哨兵张英席在向导训练营里捡到了橘猫。他抱着猫走回了哨兵所，身后尾随着袁广泉契而不舍的北极熊。

“胡浩！”张英席隔着大半个训练场喊到：“我捡到你的二级精神体了！”他说着，高高举起了猫。在场的哨兵除了郭虹旭和毛二，所有人都愣在一旁。“它溜达到向导所去了，广泉的熊还追着它跑，哈哈。”

胡浩沉默地走了过去，两侧的哨兵没法在他那张脸上看出任何表情。他跟张英席道了谢，把猫接到了手里。切布拉什卡跟上去蹭了蹭他，然后突然消失了。

“浩哥，”袁博士的温柔地声音回荡在他精神域里。“这次换我和喀秋莎去你那边住吧。”

胡浩还没搭腔，张所长又拍着他的肩膀欢乐地说道：“我刚从后勤保障部回来，广泉的婚房申请批下来了。一会越越会把钥匙给你，你们今晚就能搬进去了。这几个月你俩挤在他的单身公寓里辛苦啦。”

“谢谢所长。”胡浩再次向他道谢。随后，他转过身去，对着训练场上的哨兵们说到：“不好意思，占用了公共资源。告诉大家一个消息，我和广泉绑定了。”

“没错，”全塔的哨兵和向导精神域里响起了黑暗向导的声音，“我和浩哥已经结婚啦～”

尾声1:
“张老师，这si广泉的猫啊。”何亮辰吓得方言都冒了出来。“浩锅的二级精神体si小泰guo尔啊，我见过照片的。”何亮辰急忙打开手机相册，翻出了胡浩抱着小泰戈尔的照片。

“这是张老师组织我和上哥浩哥去参加关爱野生动物活动时照的啊。”赵一看了一眼手机说道。“这是个真的小老虎啊！塔里前年的公益任务就是这个，你那会因为跟何大闹别扭出任务受伤了，我顶替的你啊。你忘了吗？”

尾声2:
“为什么我会对你的猫毛过敏？”何宜霖把胡浩堵在路上质问道。
“我在猫毛上下了精神暗示。”胡浩耸了耸肩，“圆圆现在是我的人，你要欺负他，总得吃点苦头。”

尾声3:
“袁博士，”王上带着他一脸好奇的小向导拦住了袁广泉，对他开启了十万个为什么问答，“黑暗哨兵和黑暗向导不是非常独立互不兼容吗？为什么你和浩哥可以结合？”
“拉着浩哥多做几次就好了。”袁博士挂着天真甜美的微笑开起了赛车。加菲从他身边路过，跳起来狠狠地挠了他一把。
“不然你以为，所长一个黑暗哨兵是怎么绑定赵越哥这个哨兵的呢？”

尾声4:
“浩哥，”向导从背后紧紧地搂着哨兵，“你之前为什么不想跟我公开呢？”

哨兵沉默了许久，久到向导以为他已经睡着了。“我们的结合是个意外。”他缓缓地说着，“黑暗哨兵和黑暗向导，从来没有在一起的先例。我们精神域的通感那么强烈，精神链接又随时可能断掉。一旦我感官过载，带给你的冲击比你跟任何普通哨兵和向导链接都要剧烈。海四的任务大都九死一生，我们这样的关系，在战场上随时会变成彼此的拖累。”

向导轻轻地吻着他的脖子和肩膀，含糊地问到，“如果没有所长替我们公开，你最终会推开我吗？

”哨兵默默地在向导怀里翻了个身，直视着他的眼睛，“不会。”他坚定地说，“小白熊太好摸了，我舍不得让给别人。”

向导一贯平静温情的脸上突然露出了一丝裂痕。“不管是切布拉什卡还是喀秋莎，他们都是我。”

我可以长久地潜伏在你身边，守住你的后背，在你对我的第一丝依赖露出精神海的时候迅速扼住你的喉咙，完全占有你。我要你的猫睡在我的怀里，你的虎匍匐在我脚下。我要夜夜在你身上打上我的烙印。让死神和井都不能把你从我手里夺走。

“圆圆，你的脑子里太吵了。”哨兵笑着，吻上了向导的嘴唇。

“粘人又狡猾的小白熊，哥哥就勉为其难地收养了吧。”

The End
